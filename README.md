# Кванториум Desktop

Desktop-приложение для сотрудников Кванториума. Позволяет работать с внутренней API-системой из удобного интерфейса: вести расписание, посещаемость, мероприятия, бронирования, документы и данные сотрудников.

[![Скачать установщик](https://img.shields.io/badge/Скачать_установщик-Windows-blue?style=for-the-badge)](https://github.com/nisonya/kvant-desktop/releases/latest)

## Возможности

- Авторизация сотрудников и сохранение сессии.
- Просмотр и редактирование профиля, смена пароля.
- Управление сотрудниками и создание новых учётных записей.
- Работа с учениками, группами, посещаемостью и пикселями.
- Просмотр, добавление, изменение и удаление расписания.
- Бронирование помещений и управление списком комнат.
- Работа с мероприятиями, ответственными, документами и напоминаниями.
- Системные уведомления о запланированных мероприятиях.
- Экспорт данных в Excel.
- Автообновление desktop-приложения через сервер обновлений.

## Как начать пользоваться

1. Откройте страницу [Releases](https://github.com/nisonya/kvant-desktop/releases/latest).
2. Скачайте установщик `kvant.desktop-setup-*.exe`.
3. Запустите установщик и откройте приложение.
4. Введите адрес сервера API, который был указан при разворачивании backend-части.
5. Войдите под стартовой учётной записью администратора.

## Первый вход

После разворачивания базы данных по умолчанию создаётся администратор:

```text
Логин: rootroot
Пароль: initial123
```

Через эту учётную запись можно войти в приложение, открыть раздел сотрудников и создать учётные записи для остальных пользователей.

Важно: сразу после первого входа обязательно смените пароль администратора в профиле. Стандартный пароль нужен только для первичной настройки системы.

## Разработка


```bash

```
