# API Deployer Desktop

Electron-приложение для работы с API. Поддерживает Windows и macOS.

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm start
```

## Сборка приложения

**Для Windows:**
```bash
npm run build:win
```

**Для macOS** (только на Mac):
```bash
npm run build:mac
```

Готовые установщики появятся в папке `dist/`.
